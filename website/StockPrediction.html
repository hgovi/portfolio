Introduction¶

In this project, we aim to build a machine learning model to accurately predict the next day's price of the S&P 500 index. Our data comes from the yfinance library and contains daily stock information dating back to 1927, including opening, high, low, and close prices. Additionally, there is a volume column showing the daily traded volume. Other columns in the dataset will be removed, as they are not relevant to our analysis. We will add new features of interest during the feature engineering process.

Results¶

Our final model, if used daily since 1990 to decide whether or not to invest in the S&P 500, would have generated a net positive return. The process we followed to achieve these results is outlined below.

First, we extracted data from the yfinance library and added several key features. We created a new column called Tomorrow, which represents the closing price of the stock on the following day. We also added a binary column called Target, where a value of 0 indicates that the stock price decreased the next day, and a value of 1 indicates that the price increased. These columns were essential for training our model by providing a clear metric to predict.

Next, we built a basic model using the predictors described in the introduction. For evaluation, we defined precision score as our error metric, which measures how often we correctly predicted a price increase compared to how often it actually occurred. After establishing this baseline model, I focused on improving it through feature engineering.

To enhance the model, I introduced new columns that considered rolling averages over various time periods. The goal was to help the algorithm detect market trends, such as sustained increases or decreases, as well as potential market corrections. This improved model performed significantly better than the initial one.

In addition to feature engineering, I increased the prediction confidence threshold. Instead of predicting an increase when the model had over 50% confidence, I raised the threshold to 60%, which led to an even higher precision score. This adjustment ensured the model only predicted price increases when it was relatively certain, further improving its accuracy.

These refinements enabled us to create a reliable stock price prediction model for the S&P 500. Looking ahead, future improvements could involve adding more predictors, such as data from other stocks or international markets with different operating hours. Incorporating such factors might further enhance the model's accuracy.

def predict(train, test, predictors, model):
    model.fit(train[predictors], train["Target"])
    preds = model.predict(test[predictors])
    preds = pd.Series(preds,index=test.index, name = "Predictions")
    combined = pd.concat([test["Target"], preds], axis = 1)
    return combined

def backtest(data, model, predictors, start = 2500, step = 250):
    all_predictions = []
    
    for i in range(start, data.shape[0], step):
        train = data.iloc[0:i].copy()
        test = data.iloc[i: (i+step)].copy()
        predictions = predict(train, test, predictors, model)
        all_predictions.append(predictions)
    return pd.concat(all_predictions)

horizons = [2, 5, 60, 250, 1000]
new_predictors = []

for horizon in horizons:
    rolling_average = sp500.rolling(horizon).mean()
    
    ratio_column = f"Close_Ratio_{horizon}"
    sp500[ratio_column] = sp500["Close"] / rolling_average["Close"]
    
    trend_column = f"Trend_{horizon}"
    sp500[trend_column] = sp500.shift(1).rolling(horizon).sum()["Target"]
    
    new_predictors += [ratio_column, trend_column]

def predict(train, test, predictors, model):
    model.fit(train[predictors], train["Target"])
    preds = model.predict_proba(test[predictors])[:,1]
    preds[preds >= 0.6] = 1
    preds[preds < 0.6] = 0
    preds = pd.Series(preds,index=test.index, name = "Predictions")
    combined = pd.concat([test["Target"], preds], axis = 1)
    return combined
